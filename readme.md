# tmuxop CLI

A CLI for tmux open projects (tmuxop).

## Requirements

* [NodeJS](https://nodejs.org)
* [NPM](https://npmjs.com) or [Yarn](https://yarnpkg.com)

## Install

If you is using NPM.

```shell
$ npm i -g tmuxop
```

If you is using Yarn.

```shell
$ yarn global add tmuxop
```
